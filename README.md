# cli

Install via homebrew:
```
brew tap Selleo/cli
brew install selleo
```

or download binary for your system.

## AWS

### Dev

You can fetch secrets from AWS store parameters and run the command:

```
selleo aws dev --region eu-central-1 --path /office/dev/api npm run start
```
